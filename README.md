[ThemeOverrides]

## 🔗 API Integration: OpenAI & Claude

For power users, the toolkit exposes a lightweight API gateway that connects directly to OpenAI’s GPT-4 and Anthropic’s Claude 3.5 for **intelligent brand naming** and **tagline generation**:

- **OpenAI Integration** – Sends extracted design vectors as context for generating brand name suggestions. Example prompt: “Based on this minimalist leaf icon, suggest 10 SaaS company names under 8 characters.”
- **Claude Integration** – Used for nuanced brand voice analysis. Claude evaluates generated logos against a brand personality matrix (serious vs. playful, traditional vs. disruptive).

Both integrations are optional and toggleable in the configuration profile.
